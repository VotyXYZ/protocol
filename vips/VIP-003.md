# VIP-003: Groups under organization

This VIP adds groups in organization, aims to provide a way to set down rules for proposals.

add `groups` field to organization document:

```ts
{
  name: string;
  groups: {
    name: string;
    promise: string; // promise of proposer
    timing: {
      publicity: number;
      voting: number;
    }
    proposing_liberty: {
    }
    voting_power: {
    }
  }
  [];
}
```
